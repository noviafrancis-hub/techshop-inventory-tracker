# techshop-inventory-tracker
Java-based inventory management system for tracking products, stock levels, and updates for a small electronics business.
## Features
- Add products
- View inventory
- Search products
- Update stock
- Delete products
- Low stock alerts

## Technologies Used
- Java
- ArrayList
- OOP principles

## How to Run
1. Open in NetBeans or IntelliJ
2. Run Main.java
